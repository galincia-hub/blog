## 1. 프로젝트 개요

### 정체성
- **이름**: 모아가이드 (MoaGuide)
- **한 줄 정의**: 구글 검색 유입 기반 수익형 정보 블로그
- **톤**: 중립적 정보 전달. 브랜드 페르소나 없음. 누가 썼는지보다 정보의 정확성과 구조가 핵심.
- **콘텐츠 성격**: 행동 유발형 정보 글. 독자는 특정 행동(다운로드, 바로가기 접속, 전화번호 확인, 양식 다운로드)을 하려고 검색한 사람. 글은 그 행동을 완수하도록 안내하는 역할.
- **레퍼런스 블로그**: IT수집가 (itsujibga.com) — 디자인, 글 구조, 카테고리 구성 참조

### 수익 구조
- 애드센스 (검색 유입 → 광고 노출 → 클릭)
- 제휴마케팅 (CPA/CPS 링크 삽입)
- **수익 목표**: 월 100만 원 = 애드센스 60만 + 제휴 40만
- **트래픽 목표**: 하루 ~1,000 PV
- **필요 글 수**: 50~80개 (안정 유입 기준)

### 수익화 요소 (나중에 추가)
- 애드센스 광고 영역: 승인 후 삽입 (_includes/adsense.html)
- 제휴마케팅 링크: 글 유형별 자연스러운 배치
- 팝업/배너: 별도 생성
- **현재 단계에서는 광고 요소 없이 순수 콘텐츠와 사이트 구조에만 집중**

## 4. 사이트 구조

### 페이지 목록
- **메인 (/)**: 전체 글 카드 리스트 + 상단 카테고리 네비게이션
- **카테고리 페이지 (/it/, /life/, /finance/, /health/, /review/)**: 해당 카테고리 글만 필터링된 목록
- **글 페이지 (/posts/slug/)**: 개별 글 본문
- **About (/about/)**: 블로그 소개 (애드센스 승인 필수)
- **Privacy (/privacy/)**: 개인정보처리방침 (애드센스 승인 필수)

### 메인 페이지 구성
```
[상단 네비게이션]
  로고(모아가이드) | IT 정보 | 생활 정보 | 금융 정보 | 건강 정보 | 제품 추천

[글 카드 리스트]
  썸네일 | 카테고리 뱃지 | 제목 | 요약 2줄 | 날짜
  (IT수집가와 동일한 카드 레이아웃)

[페이지네이션]
  1 2 3 ... 다음 →
```

### 글 페이지 구성
```
[상단 네비게이션] (동일)

[카테고리 뱃지] | [날짜]
[제목 (h1)]
[요약/서브타이틀]

[본문]
  소제목(h2) → 설명 → 표/리스트 → 다음 소제목 반복
  FAQ 섹션 (있는 경우)

[관련 글 추천] (같은 카테고리 글 3~4개)

(광고 영역은 나중에 추가)
```

### 디자인 스펙
- **배경**: 흰색 (#FFFFFF)
- **텍스트**: 다크그레이 (#333333)
- **포인트 컬러**: 카테고리별 색상 활용
- **폰트**: 시스템 폰트 (Noto Sans KR 또는 브라우저 기본)
- **레이아웃**: 단일 컬럼, 최대 너비 800px 내외
- **카드**: 썸네일 좌측 + 텍스트 우측 (IT수집가 스타일)
- **모바일**: 반응형 (카드 세로 스택)

- **장식**: 반투명 원형 도형 (우측 상단, 우측 하단, 좌측 하단)

3. .md frontmatter의 `thumbnail` 필드에 경로 자동 연결

thumbnail: /assets/images/thumbnails/파일명.png

tags: [키워드1, 키워드2, 키워드3]

## 도입 (2~3문장)

## 소제목 1
설명 + 표 또는 리스트

## 소제목 2

## 소제목 3 (필요 시)
...

## 자주 묻는 질문 (FAQ) — 있는 경우

## 9. SEO/AEO 기본 인프라

### Jekyll 플러그인
- `jekyll-sitemap`: sitemap.xml 자동 생성
- `jekyll-seo-tag`: 메타 태그 자동 삽입
- `jekyll-feed`: RSS 피드 자동 생성

### Schema.org (JSON-LD)
글 카테고리에 따라 적합한 스키마를 자동 선택하여 _includes/schema.html에서 삽입:
- **Article**: 모든 글 공통
- **FAQPage**: FAQ 섹션이 있는 글
- **SoftwareApplication**: 유틸리티 다운로드 글
- **HowTo**: 설치/사용법 가이드 글

### 기본 SEO 설정
- robots.txt: 크롤링 허용
- Google Search Console: 사이트 등록 (배포 후)
- meta title: "키워드 | 모아가이드" 형식
- meta description: 글 요약 1~2문장
- URL 구조: /카테고리/글-슬러그/ (한글 URL 지양, 영문 슬러그 사용)

- 썸네일 10개 정상 표시
- **글 10개의 품질과 방향성이 만족스러운지 확인**
