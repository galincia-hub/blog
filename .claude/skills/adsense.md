# 애드센스 승인 스킬 (adsense.md)

## 적용 시점
이 스킬은 애드센스 승인이 완료될 때까지 모든 글 작성에 강제 적용한다.
사용자가 **"애드센스 승인됨"** 이라고 알리면 강제 적용을 해제한다.

---

## 2026년 애드센스 승인 핵심 요건

### 사이트 구조 필수 요소
- [x] HTTPS 적용 (Netlify 자동 SSL)
- [x] 반응형 디자인 (모바일 대응)
- [x] About 페이지 (운영자 신원, 사이트 목적 명시)
- [x] Privacy Policy 페이지
- [x] Contact 페이지 (실제 동작하는 이메일)
- [x] 네비게이션 메뉴 (사이트 구조 명확)
- [x] 시멘틱 HTML (`<nav>`, `<main>`, `<article>`)

### 콘텐츠 수량 기준
- 최소 15~20개 이상 글 필요 (현재 10개 → 추가 필요)
- 각 글 최소 1,000자 이상 (권장 1,500~3,000자)
- 800자 미만 글 = 자동 거절 위험

### 독창성 기준
- 다른 사이트 복사/재가공 금지
- 중복 콘텐츠 절대 금지
- AI 생성이라도 자연스러운 한국어로 편집 필수

### 정보성 기준
- 각 글이 명확한 주제 + 사용자 정보적 필요 충족
- "이 글을 읽고 나면 독자가 행동을 완수할 수 있는가?" 자기 검증
- YMYL(건강·금융) 글은 과장 주장 금지

### AI 콘텐츠 대응
- 구글은 AI 콘텐츠를 차별하지 않음 (2024년 이후)
- 단, 비자연스러운 문장 구조·높은 중복률은 거절 사유
- 기계적 반복 패턴 제거
- 자연스러운 한국어 문체 유지

### 금지 콘텐츠
- 성인/폭력/불법 행위 조장
- 저작권 침해 자료
- 클릭 유도 문구 ("광고 클릭 부탁드립니다")
- 다른 사이트 콘텐츠 복사/스크래핑

## 글 작성 시 자기 검증 체크리스트 (애드센스 승인 전 필수)

글 생성 완료 후 아래 체크리스트를 반드시 실행한다.
**하나라도 미통과 시 수정 후 저장한다.**

- [ ] 1,000자 이상인가?
- [ ] 독창적 내용인가? (복사 아닌 자체 작성)
- [ ] 명확한 주제가 있고, 검색 의도에 답하는가?
- [ ] 소제목(h2)이 3개 이상인가?
- [ ] 표가 1개 이상 있는가?
- [ ] 이미지에 ALT 텍스트가 있는가?
- [ ] 자연스러운 한국어인가? (AI투/번역투 없음)
- [ ] 금지 콘텐츠에 해당하지 않는가?
- [ ] CTA 버튼이 2개 이상인가?
- [ ] FAQ 섹션이 있는가?
- [ ] last_updated 날짜가 있는가?
