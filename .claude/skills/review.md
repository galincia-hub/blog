# 리뷰/검수 스킬 (review.md)

## 팩트 검증 원칙
- 공식 사이트 URL은 실제 접근 가능한지 확인
- 전화번호는 최신 정보인지 확인
- 가격/수수료/기간 정보는 "기준 날짜" 명시
- 정부 서비스 URL은 gov.kr 도메인 여부 확인

## 글 품질 검수 체크리스트
- [ ] 제목에 핵심 키워드 포함
- [ ] 첫 문단이 검색 의도에 직접 답하는가
- [ ] 소제목(h2) 3개 이상
- [ ] 표 1개 이상
- [ ] 이미지 ALT 텍스트 있는가
- [ ] CTA 버튼 2개 이상 (상단/하단)
- [ ] FAQ 섹션 있는가
- [ ] last_updated 날짜 있는가
- [ ] 자연스러운 한국어인가 (AI투/번역투 없음)
- [ ] 1,000자 이상인가
- [ ] 내부 링크 또는 related-posts 작동하는가
