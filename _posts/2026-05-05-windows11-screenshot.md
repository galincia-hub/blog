윈도우11에서 화면을 캡처하려면 **Win + Shift + S** 단축키가 가장 빠릅니다. 전체 화면, 특정 영역, 창 단위로 캡처할 수 있으며, 자동 저장이 필요하면 **Win + PrtSc** 키를 사용합니다.

## 윈도우11 화면 캡처 방법 5가지 비교

| 방법 | 단축키 | 특징 | 저장 방식 |
|------|--------|------|---------|
| 캡처 도구 (부분 캡처) | Win + Shift + S | 영역·창·전체 선택 가능 | 클립보드 → 수동 저장 |
| Print Screen | PrtSc | 전체 화면 캡처 | 클립보드 |
| 자동 저장 캡처 | Win + PrtSc | 전체 화면 자동 저장 | 사진\스크린샷 폴더 |
| 캡처 도구 앱 | 시작 메뉴 검색 | 지연 캡처·주석 기능 | 직접 저장 |
| 게임 바 | Win + G | 게임·영상 화면 캡처 | 동영상\캡처 폴더 |

---

## 방법 1: Win + Shift + S (가장 많이 쓰는 방법)

부분 영역을 자유롭게 선택해서 캡처할 수 있는 가장 실용적인 방법입니다.

1. **Win + Shift + S** 키를 동시에 누릅니다.
2. 화면이 어두워지면서 상단에 캡처 모드 선택 바가 나타납니다.
3. 원하는 모드를 선택합니다.
   - **사각형 캡처**: 드래그로 영역 지정
   - **자유형 캡처**: 마우스로 원하는 모양 그리기
   - **창 캡처**: 특정 창 전체 캡처
   - **전체 화면 캡처**: 전체 화면 즉시 캡처
4. 캡처 후 오른쪽 하단에 알림이 뜹니다. 클릭하면 편집 화면이 열립니다.
5. **Ctrl + S**로 저장하거나, **Ctrl + C**로 복사해 붙여넣기 할 수 있습니다.

> **팁**: 캡처한 이미지는 클립보드에 자동 저장됩니다. 카카오톡, 워드, 포토샵 등에서 바로 **Ctrl + V**로 붙여넣기 가능합니다.

---

## 방법 2: Win + PrtSc (자동 저장)

전체 화면을 캡처하면서 파일로 자동 저장됩니다. 반복 캡처가 필요한 작업에 유용합니다.

1. **Win + PrtSc** 키를 동시에 누릅니다.
2. 화면이 잠깐 어두워지면 캡처 완료입니다.
3. 저장 위치: **내 PC → 사진 → 스크린샷** 폴더

파일명은 `스크린샷 (1).png`, `스크린샷 (2).png` 순서로 자동 지정됩니다.

---

## 방법 3: PrtSc 단독 (클립보드 복사)

가장 간단한 방법으로, 전체 화면을 클립보드에 복사합니다.

1. 키보드의 **PrtSc** (또는 Print Screen) 키를 누릅니다.
2. 클립보드에 저장됩니다.
3. 그림판, 워드, 카카오톡 등에서 **Ctrl + V**로 붙여넣기 합니다.
4. 그림판에서 붙여넣기 후 **Ctrl + S**로 파일 저장이 가능합니다.

---

## 방법 4: 캡처 도구 앱 (지연 캡처·주석)

팝업 메뉴처럼 나타났다 사라지는 화면을 캡처하거나, 주석을 추가해야 할 때 유용합니다.

1. 시작 메뉴에서 **"캡처 도구"** 검색 후 실행합니다.
2. **새로 만들기** 클릭 후 캡처 모드 선택
3. **지연** 기능: 3~10초 후 캡처 예약 가능 (툴팁, 드롭다운 메뉴 캡처에 유용)
4. 캡처 후 펜·형광펜·지우개로 주석 추가 가능
5. 파일 → 다른 이름으로 저장 (PNG, JPG, GIF 지원)

---

## 방법 5: Win + G 게임 바 (영상·게임 캡처)

게임 플레이 중이나 동영상 재생 화면 캡처에 적합합니다.

1. **Win + G** 키를 눌러 게임 바를 실행합니다.
2. 캡처 패널에서 카메라 아이콘(📷) 클릭하면 스크린샷 저장
3. 저장 위치: **내 PC → 동영상 → 캡처** 폴더

> **주의**: 게임 바는 일부 앱이나 데스크탑에서 작동하지 않을 수 있습니다.

## 자주 묻는 질문 (FAQ)

**Q. Win + Shift + S가 작동하지 않아요.**
A. 키보드 언어 설정이 영문(ENG)인지 확인하세요. 한글 입력 상태에서는 일부 단축키가 작동하지 않습니다. 작업 표시줄 우측 언어 표시를 클릭해 영문으로 전환 후 시도해보세요.

**Q. 캡처한 파일이 어디에 저장되나요?**
A. Win + PrtSc로 캡처하면 `내 PC → 사진 → 스크린샷` 폴더에 자동 저장됩니다. Win + Shift + S는 클립보드에만 저장되므로 직접 저장해야 합니다.

**Q. 특정 창만 캡처하는 단축키가 있나요?**
A. **Alt + PrtSc** 키를 누르면 현재 활성화된 창만 클립보드에 캡처됩니다. 그림판에서 붙여넣기 후 저장하세요.

**Q. 캡처 도구가 시작 메뉴에 없어요.**
A. 시작 메뉴 검색창에 "캡처 도구"를 입력하면 나타납니다. 없다면 **Microsoft Store**에서 "Snipping Tool"을 검색해 설치할 수 있습니다.

{% include cta-button.html url="https://apps.microsoft.com/detail/9mz95kl8mr0l" text="캡처 도구 Microsoft Store에서 받기" type="download" %}

상황에 따라 단축키를 구분해 사용하면 편리합니다. 빠른 부분 캡처는 **Win + Shift + S**, 자동 저장이 필요하면 **Win + PrtSc**, 지연 캡처나 주석이 필요하면 **캡처 도구 앱**을 활용하세요.
