건강보험 자격득실확인서는 건강보험 가입 이력을 증명하는 서류로, 대출 심사, 보험 가입, 취업 시 자주 제출합니다. 국민건강보험공단 홈페이지에서 즉시 무료로 발급받을 수 있습니다.

{% include cta-button.html url="https://www.nhis.or.kr" text="건강보험공단 바로가기" type="link" %}

## 발급 경로 비교

| 발급 방법 | 소요 시간 | 수수료 | 비고 |
|---------|---------|--------|------|
| 건강보험공단 홈페이지 | 즉시 | 무료 | 가장 빠름 |
| 정부24 | 즉시 | 무료 | 통합 서비스 |
| 건강보험공단 앱 | 즉시 | 무료 | 모바일 |
| 국민건강보험공단 지사 방문 | 당일 | 무료 | 신분증 지참 |
| 무인민원발급기 | 즉시 | 무료 | 지사·주민센터 |

## 국민건강보험공단 홈페이지 발급

1. **nhis.or.kr** 접속
2. 오른쪽 상단 **로그인** (공동인증서, 간편인증)
3. 메인 메뉴 **민원여기요 → 개인민원** 클릭
4. 검색창에 **"자격득실확인서"** 입력
5. **"건강보험 자격득실 확인서"** 선택
6. 조회 기간 설정 후 **발급** 클릭
7. PDF로 저장 또는 인쇄

## 정부24에서 발급

1. **gov.kr** 접속
2. 검색창에 **"건강보험 자격득실확인서"** 입력
3. **본인인증** 완료
4. 조회 기간 설정
5. **발급** 후 PDF 저장

## 모바일 발급 (건강보험공단 앱)

1. **The건강보험** 앱 설치 (iOS/Android)
2. 로그인
3. **증명서 발급** 메뉴 선택
4. 자격득실확인서 선택 후 발급

## 자격득실확인서 내용 설명

발급된 서류에는 다음 정보가 포함됩니다.

| 항목 | 내용 |
|------|------|
| 자격 구분 | 직장가입자 / 지역가입자 / 피부양자 |
| 자격 취득일 | 건강보험 가입 시작일 |
| 자격 상실일 | 해당 직장 퇴직 또는 피부양자 자격 상실일 |
| 사업장 명칭 | 직장가입자의 경우 회사명 |

## 자주 묻는 질문 (FAQ)

**Q. 퇴직 후 건강보험 납부 이력도 확인할 수 있나요?**
A. 네, 자격득실확인서에 과거 직장 가입 이력이 모두 포함됩니다. 최대 10년치 이력 조회가 가능합니다.

**Q. 가족(피부양자) 이력도 발급받을 수 있나요?**
A. 본인 명의로 로그인 시 본인 이력만 조회됩니다. 피부양자로 등록된 가족의 이력은 해당 가족이 직접 발급받아야 합니다.

**Q. 인증서 없이 발급받을 수 있나요?**
A. 카카오톡·네이버·PASS 등 간편인증을 이용하면 공동인증서 없이도 발급 가능합니다.

nhis.or.kr 또는 정부24에서 간편인증으로 로그인하면 5분 이내에 즉시 발급받을 수 있습니다. PDF를 저장해두면 이후 제출 시 편리합니다.
