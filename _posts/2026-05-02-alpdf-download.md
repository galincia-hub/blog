3. **"무료 다운로드"** 버튼을 클릭합니다.
4. 설치 파일 실행 후 안내에 따라 설치합니다.

## PDF 변환 사용 방법

### Word/Excel → PDF 변환
1. 알PDF 실행 후 **"파일 변환"** 탭 클릭
2. 변환할 파일을 드래그하거나 파일 추가
3. 출력 형식에서 PDF 선택
4. **"변환 시작"** 클릭

### PDF → Word 변환
1. **"PDF 변환"** 탭 선택
2. 변환할 PDF 파일 추가
3. 출력 형식에서 Word(.docx) 선택
4. 변환 시작 (무료 버전: 월 5회 제한)

## PDF 합치기 방법

여러 PDF 파일을 하나로 합칠 때 유용합니다.

1. **"PDF 합치기"** 기능 선택
2. 합칠 파일들을 순서대로 추가
3. 파일 순서 드래그로 조정
4. **"합치기"** 실행 후 저장 경로 지정

## 자주 묻는 질문 (FAQ)

**Q. 알PDF와 Adobe Acrobat의 차이는?**
A. Adobe Acrobat은 유료지만 더 강력한 편집 기능을 제공합니다. 알PDF는 기본적인 변환·합치기·나누기 작업을 무료로 처리하기에 충분합니다.

**Q. 변환 품질이 떨어지지 않나요?**
A. 기본적인 텍스트·표 구조는 잘 유지됩니다. 복잡한 레이아웃이나 특수 폰트가 포함된 문서는 변환 후 확인이 필요할 수 있습니다.

**Q. Mac에서 사용할 수 있나요?**
A. 알PDF는 Windows 전용입니다. Mac에서는 기본 미리보기 앱이나 PDF Expert를 추천합니다.
