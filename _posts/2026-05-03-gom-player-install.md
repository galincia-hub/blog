곰플레이어(GOM Player)는 국내에서 가장 많이 사용하는 무료 동영상 재생 프로그램입니다. 공식 사이트에서 무료로 다운로드할 수 있으며, 아래에서 설치 과정을 단계별로 안내합니다.

## 곰플레이어 사양 및 특징

| 항목 | 내용 |
|------|------|
| 가격 | 무료 (광고 포함) / GOM Player Plus (유료, 광고 없음) |
| 지원 OS | Windows 7/8/10/11 |
| 지원 형식 | AVI, MP4, MKV, MOV, WMV 등 대부분 형식 |
| 특징 | 코덱 내장, 자막 자동 검색, 360° VR 재생 |

## 공식 다운로드 방법

1. 곰앤컴퍼니 공식 사이트에 접속합니다.
2. 메인 화면의 **"무료 다운로드"** 버튼을 클릭합니다.
3. 설치 파일(.exe)이 다운로드됩니다.
4. 다운로드한 파일을 실행합니다.

> **주의**: 검색 엔진에서 "곰플레이어 다운로드"를 검색하면 비공식 사이트가 상단에 노출될 수 있습니다. 반드시 공식 사이트(gom.com)에서 받으세요.

## 설치 시 주의사항

곰플레이어 설치 과정에서 추가 프로그램 설치를 권유하는 화면이 나옵니다.

- **설치 중 체크박스 확인**: "추가 프로그램 설치" 또는 "시작 페이지 변경" 체크박스를 **해제**하고 진행하세요.
- 원치 않는 프로그램이 함께 설치되는 것을 방지할 수 있습니다.

## 설치 후 기본 설정

설치 완료 후 처음 실행 시 설정을 잡아두면 편리합니다.

1. **환경설정(F5)** 열기
2. **재생** 탭: 기본 자막 언어를 한국어로 설정
3. **화면** 탭: 재생 시 자동으로 창 크기 조정 여부 설정
4. **파일 연결**: 원하는 파일 형식을 곰플레이어와 연결

## 자주 묻는 질문 (FAQ)

**Q. 곰플레이어 무료 버전에 광고가 있나요?**
A. 네, 무료 버전은 실행 시 광고가 표시됩니다. 광고 없는 사용을 원하면 GOM Player Plus(유료)를 구매하거나, 광고 표시 시간 동안 기다리면 됩니다.

**Q. 4K 영상도 재생되나요?**
A. 네, 최신 버전의 곰플레이어는 4K UHD 영상 재생을 지원합니다. 하드웨어 가속(GPU 디코딩)도 지원합니다.

**Q. Mac에서 사용할 수 있나요?**
A. 곰플레이어는 Windows 전용입니다. Mac에서는 VLC 또는 IINA를 추천합니다.
